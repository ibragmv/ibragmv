## `operating_principles`

| Principle | Meaning |
| --- | --- |
| **Build with taste** | Interfaces should feel intentional, not assembled from leftovers. |
| **Respect the future** | Structure code so the next version is easier, not harder. |
| **Kill legacy gravity** | No shortcuts, no zombie code, no “we will clean it later”. |
| **Ship alive things** | If it can move, update, react, or surprise, it probably should. |
